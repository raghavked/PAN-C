# 🔧 FIX: React Version Mismatch Error in Expo Go

**ERROR**: 
```
Incompatible React versions:
- react: 19.2.6
- react-native-renderer: 19.1.0
```

**CAUSE**: React and React Native have different versions installed  
**SOLUTION**: Downgrade React to match React Native, or upgrade React Native

---

## QUICK FIX (Recommended)

### Option A: Downgrade React (Fastest)

```bash
cd rn

# Remove current node_modules
rm -rf node_modules

# Downgrade react to 19.1.0 to match react-native-renderer
npm install react@19.1.0 --save

# Reinstall all dependencies
npm install

# Clear Metro cache
npm start -- --reset-cache

# Or if using expo start:
npx expo start --clear
```

---

### Option B: Upgrade React Native (If You Need Latest React)

```bash
cd rn

# Remove node_modules
rm -rf node_modules

# Check current React Native version
npm list react-native

# Upgrade React Native to latest
npm install react-native@latest

# Reinstall dependencies
npm install

# Clear cache
npx expo start --clear
```

## STEP-BY-STEP FIX

### Step 1: Go to RN Directory

```bash
cd rn
```

### Step 2: Check Current Versions

```bash
npm list react react-native

# Should show what's currently installed
```

### Step 3: Remove Node Modules

```bash
# This clears cached packages
rm -rf node_modules package-lock.json

# Verify deletion
ls node_modules 2>/dev/null && echo "⚠️  Still there" || echo "✅ Removed"
```

### Step 4: Fix the Version Mismatch

**Choose ONE approach:**

**Approach A - Keep React 19.1.0 (Recommended)**
```bash
# Install exact matching versions
npm install react@19.1.0 react-native@19.1.0

# Then install all other dependencies
npm install

echo "✅ Versions aligned to 19.1.0"
```

**Approach B - Use Latest Compatible**
```bash
# Get latest stable versions that work together
npm install react@latest react-native@latest

# Install all dependencies
npm install

echo "✅ Updated to latest compatible versions"
```

### Step 5: Clear Metro Cache

```bash
# Clear Metro bundler cache
npx expo start --clear

# Or manually delete cache
rm -rf ~/.expo
```

### Step 6: Verify Fix

```bash
# Check versions are now the same
npm list react react-native

# Should show:
# ├── react@19.1.0
# └── react-native@19.1.0

echo "✅ Versions matched!"
```

## VERIFY IN PACKAGE.JSON

After fixing, check `rn/package.json`:

```json
{
  "dependencies": {
    "react": "^19.1.0",
    "react-native": "^19.1.0",
    "react-native-renderer": "^19.1.0",
    ...
  }
}
```

All should be same major.minor version (19.1.x is fine, 19.1.0 vs 19.2.0 is NOT fine)

---

## WHAT EACH VERSION CONTROLS

| Package | Purpose |
|---------|---------|
| `react` | React core library |
| `react-native` | React Native framework |
| `react-native-renderer` | Converts React to native code |

**They MUST match because** React Native renderer depends on React's internal APIs which change between versions.
